\abstract

แบบตัวอักษรดั้งเดิมในชุด นรสีห์ และ ครุฑ
นั้นได้ถูกออกแบบมาสำหรับการเขียนเอกสารภาษาไทยเป็นหลัก
ในการใช้แบบตัวอักษรนรสีห์และครุฑในการเขียนเอกสารบน \LaTeX{}
ที่มีการใช้ภาษาอังกฤษและสมการทางคณิตศาสตร์ 
โดยดั้งเดิมนั้นผู้ใช้มักจะพบเรื่องการกระโดดของระหว่างรูปแบบตัวอักษรภาษาอังกฤษ
ตัวอักษรทางคณิตศาสตร์ และ ตัวอักษรภาษาอังกฤษ ในชุดของนรสีห์และครุฑ

ทางออกที่ได้นำเสนอนี้เป็นการใช้ความสามารถทาง virtual font ของ \TeX{}
ซึ่งทำให้มีโอกาสที่จะรวมส่วนตัวอักษรภาษาไทย
กับแบบตัวอักษรภาษาอังกฤษที่เป็นที่นิยมใช้กันแพร่หลายได้สะดวก


\section{นรสีห์}

{\usefont{LTH}{norasi}{m}{n}
แบบตัวอักษรในชุดนรสีห์ (norasi) นั้น บน \LaTeX{}
ผู้ใช้มีรูปแบบตัวอักษรให้เลือกทั้งหมดสี่แบบด้วยกัน
%
\begin{enumerate}
  \item \textup{แบบธรรมดา (regular)}
  \item \textsl{แบบเอียง (oblique)}
  \item \textbf{แบบหนา (bold)}
  \item \textbf{\textsl{แบบหนาเอียง (bold--oblique)}}
\end{enumerate}
%
เมื่อเทียบกับแบบตัวอักษรในชุด Times จะเห็นได้ชัดว่ามีลักษณะคล้ายกันมาก\footnote
{คาดว่าน่าจะคล้ายกับ Times New Roman มากกว่า Times}
%
\begin{itemize}
  \item This is an English text set in Norasi;
        \textsl{plus a very good phrase in oblique}.
  \item {\usefont{T1}{ptm}{m}{n}\selectfont
         This is an English text set in Times;
         \textit{plus a very good phrase in italic}.}
\end{itemize}
%
ย่อหน้าในเรื่องของแบบอักษรในชุดนรสีห์นี้ได้ใช้ตัวอักษรชุดนรสีห์ในการจัดทำ
}


\section{ครุฑ}

{\usefont{LTH}{garuda}{m}{n}
แบบตัวอักษรในชุดครุฑ (garuda) นั้น บน \LaTeX{}
ผู้ใช้มีรูปแบบตัวอักษรให้เลือกทั้งหมดสี่แบบเช่นเดียวกับชุดนรสีห์
%
\begin{enumerate}
  \item \textup{แบบธรรมดา (regular)}
  \item \textsl{แบบเอียง (oblique)}
  \item \textbf{แบบหนา (bold)}
  \item \textbf{\textsl{แบบหนาเอียง (bold--oblique)}}
\end{enumerate}
%
เมื่อเทียบกับแบบตัวอักษรในชุด Helvetia
 จะเห็นได้ชัดว่ามีลักษณะคล้ายกันมาก\footnote
 {คาดว่าน่าจะคล้ายกับ \textsf{Arial} มากกว่า \textsf{Helvetica}, จากการสังเกตุของตัวอักษร a, t, R,
 และอื่นๆ}
%
\begin{itemize}
  \item This is an English text set in Garuda;
        \textsl{plus a very good phrase in oblique}.
      \item {\usefont{T1}{phv}{m}{n}\fontsize{9}{12}\selectfont
         This is an English text set in Helvetica;
         \textsl{plus a very good phrase in oblique}.}
\end{itemize}
%
ย่อหน้าในเรื่องของแบบอักษรในชุดครุฑนี้ได้ใช้ตัวอักษรชุดครุฑในการจัดทำ
}

\section{ทางเลือก}

เนื่องจากการออกแบบดั้งเดิมของนรสีห์และ\textsf{ครุฑ}นั้นมีความคล้ายคลึง
และดูกลมกลืนกับ Times และ \textsf{Helvetica} ที่เป็นชุดอักษรแบบ 
Postscript Type1 ที่เป็นนิยมใช้กันแพร่หลายกอปรกับ \LaTeX{} มี package
\textsf{mathtime} และ \textsf{mathptmx} สำหรับสนับสนุกการใช้ Times
ในสมการทางคณิตศาสตร์ ในขั้นต้นนี้จึงเป็นการทดลองผสมนรสีห์เข้ากับ Times
และผสมครุฑเข้ากับ \textsf{Helvetica}

\subsection{\textsc{Fontinstall}}

\textsc{Fontinstall} เป็น package ที่ออกแบบมาสำหรับการติดตั้งแบบตัวอักษรสำหรับ 
\TeX{} และ \LaTeX{}
 โดยที่ดั้งเดิมนั้นจะถูกใช้สำหรับการติดตั้งแบบอักษรภาษาอังกฤษในแบบ Type1 
(เช่น Times และ \textsf{Helvetica}) ในกรณีของการผสม นรสีห์ และ \textsf{ครุฑ}
เข้ากับ Times และ \textsf{Helvetica} ผู้เขียนได้ทดลองสร้างชุดรับรองการติดตั้งแบบอักษรภาษาไทยสำหรับ \textsc{Fontinstall} 

ในขั้นแรกนี้หลังจากผสม Times และ นรสีห์ 
รูปแบบตัวอักษรมีให้เลือกเพิ่มขึ้นเป็นแปดแบบด้วยกัน
%
\begin{enumerate}
  \item \textup{แบบธรรมดา (regular)}
  \item \textit{แบบเอียง (italic)}
  \item \textsl{แบบเอียง (oblique)}
  \item \textsc{แบบ Smallcaps}
  \item \textbf{แบบหนา (bold)}
  \item \textbf{\textit{แบบหนาเอียง (italic)}}
  \item \textbf{\textsl{แบบหนาเอียง (bold--oblique)}}
  \item \textbf{\textsc{แบบหนา Smallcaps }}
\end{enumerate}
%
ส่วนการผสม \textsf{Helveltica} และ \textsf{ครุฑ} 
รูปแบบตัวอักษรมีให้เลือกเพิ่มขึ้นเป็นหกแบบด้วยกัน
%
\begin{enumerate}
  \item \textsf{\textup{แบบธรรมดา (regular)}}
  \item \textsf{\textsl{แบบเอียง (oblique)}}
  \item \textsf{\textsc{แบบ Smallcaps}}
  \item \textsf{\textbf{แบบหนา (bold)}}
  \item \textsf{\textbf{\textsl{แบบหนาเอียง (bold--oblique)}}}
  \item \textsf{\textbf{\textsc{แบบหนา Smallcaps }}}
\end{enumerate}
%

\section{ความกลมกลืน}

ในการรวม \textsf{ครุฑ เข้ากับ} นรสีห์ในส่วนของภาษาไทยนั้น
นับได้ว่ามีความกลมกลืนกันดีอยู่ในระดับหนึ่ง สามารถที่จะใช้งานในลักษณะนี้ได้เช่น

\vspace{3mm}

{\par \centering\noindent\parbox{60mm}{
\noindent\textsf{\textbf{\large หัวเรื่อง}}

\vspace{3mm}

ข้อความต่างๆ ในบทความ

\vspace{3mm}
} \par}
%
ส่วนในเรื่องของความสูง\textsf{และความหนานั้น ครุฑ} และ นรสีห์ เข้ากันได้ดี
ส่วนภาษาอังกฤษนั้น Postscript Type1 Times font กลืนกับ นรสีห์ 
ได้ดีภายหลังการปรับ
\emph{scaling parameter} ใน \textsc{Fontinstall} ส่วน 
\textsf{Helvetica นั้น เป็น font ที่ออกจะใหญ่กว่า} Times เล็กน้อย
\textsf{จึงต้องปรับ \emph{scaling parameter} ลงมา ให้เท่าๆ กับ} Times;
\textsf{Roughly, text set in Helvetica} and Times always looks
different anyway.

ข้างล่างนี้เป็นการจำลองการเขียนสมการทางคณิตศาตร์จากหนังสือ
\emph{The \LaTeX{} Companion}~\cite{latexcompanion}

\subsection{Multiple integral signs}

คำสั้ง \texttype{\textbackslash iiint} และ \texttype{\textbackslash iiiint}
สร้างเครื่องหมาย integral สามชั้น และ สี่ชั้น โดยที่ช่องว่างใน text style 
ถูกปรับให้สวยงามเช่น $\iiint_A f(x,y,z)\,dx\,dy\,dz$ และ
$\iiiint_A f(x,y,z)\,dw,\,dx\,dy\,dz$ รวมทั้ง display style เช่น
%
\begin{eqnarray}
\iiint_A f(x,y,z)\,dx\,dy\,dz\\
\iiiint_A f(x,y,z)\,dw,\,dx\,dy\,dz
\end{eqnarray}

\subsection{Binomial expressions}

สำหรับกลุ่ม binomial expressions เช่น $\binom{n}{k}$ มีคำสั่ง
\texttype{\textbackslash binom},
\texttype{\textbackslash dbinom}, และ
\texttype{\textbackslash tbinom} เป็นชุดคำสั่งเฉพาะ ในกรณีของ
\texttype{\textbackslash binom} นั้นเป็นคำย่อของ
\texttype{\textbackslash fracwithdelims()[0pt]}
\begin{eqnarray}
  \sum_{\gamma \in \Gamma_C} I_{\gamma} &=& 
  2^k - \binom{k}{1} 2^{k-1} + \binom{k}{2} 2^{k-2} \\
  && + \cdots + (-1)^l \binom{k}{l} 2^{k-l} + \cdots + (-1)^k \notag \\
  &=& (2-1)^k = 1 \notag
\end{eqnarray}

\subsection{Split equations}

The \texttype{split} environment provides no numbering because it is
inteded to be used inside some other displayed equation structure, such
as \texttype{equation}, \texttype{align}, or \texttype{gather}. These
outer environments will provide the numbering.

\begin{equation}
  \begin{split}
    (a+b)^4 
    &= (a+b)^2 (a+b)^2 \\
    &= (a^2+2ab+b^2)(a^2+2ab+b^2) \\
    &= a^4 + 4a^3b + 6a^2b^2 + 4ab^3 + b^4 \\
  \end{split}
\end{equation}

ในตัวอย่างข้างต้นทั้งหมดนั้น สมการทางคณิตศาสตร์ได้ใช้ package \textsf{mathtime}
ร่วมกับ \textsf{belleek} ที่เป็น free replacement of commercial mathtime 
ในการเขียน

\section{ผลพลอยได้}

จากการใช้ \textsc{Fontinstall} ทำให้สามารถสร้าง ligature rules สระอุ และ สระอู
สำหรับใช้กับตัวอักษร ป ฟ และ ฝ ในกรีสำหรับคำว่าปู่ 
({\usefont{LTH}{norasi}{m}{n} ปู่ ในนรสีห์ดั้งเดิม})
โดยการเขียน \mbox{ป-- ู-- ่่}
\textsf{หรือ ปู้}
({\usefont{LTH}{garuda}{m}{n} ปู้ ในครุฑดั้งเดิม})
 ทั้งนี้จากความคิดของคุณเทพพิทักษ์ และคุณอานนท์แห่ง 
{\texttype{linux.thai.net}} โดยที่เป็นการขโมยตัวอักษรในตำแหน่ง \texttype{0xDD}
และ \texttype{0xDE} ที่ว่างอยู่ใช้ไปก่อน

ligaure ภาษาอังกฤษรวมทั้ง fi fl ligatures
({\usefont{LTH}{norasi}{m}{n} fi fl ในนรสีห์ดั้งเดิม})
และตัวอักษรทั้งหมดที่อยู่ใน 8r encoding ที่มีรหัส
 \texttype{0x00}--\texttype{0x7F} เป็นผลพลอยได้เพิ่ม

เนื่องจากการเลือกใช้แบบตัวอักษรในรูปแบบ Adobe Type1 ทั้งหมด การสร้างเอกสาร
 \textsc{pdf} (Portable Document Format) จึงทำได้โดยสวยงาม

\section{สรุป}

สนุกครับกับการเล่นกับ \textsc{Fontinstall} ในภายหน้าการเขียนเอกสารภาษาไทยด้วย 
\TeX/\LaTeX\ คงจะทำได้สาวยงามขึ้นในเรื่องของแบบตัวอักษร ส่วนการใช้งานง่ายๆ
โดยที่ไม่จำเป็นต้องมีความรู้เรื่อง \TeX/\LaTeX\ มากนัก \LyX\
เป็นทางเลือกที่ง่ายที่สุดทางหนึ่ง
